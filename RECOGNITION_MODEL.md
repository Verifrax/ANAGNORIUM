# Recognition model

A recognition object answers one bounded question:

Which claim, under which governing law version, accepted epoch, authority basis, execution basis, and verification basis, is admissible for terminal recognition?

Recognition is lawful only when:

- one claim class is identified
- one governing law version is identified
- one accepted epoch is identified
- one authority object or authority reference is identified where required
- one execution receipt or equivalent execution basis is identified where required
- one verification result or equivalent evaluation basis is identified where required
- the recognition basis is explicit
- the recognition result is explicit

Recognition does not assign burden.
Recognition does not assign remedy.
Recognition does not define closure.
